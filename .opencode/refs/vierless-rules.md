# VIERLESS Design Rules

**Reference:** https://vierless.de/en
**Style:** Premium dark agency, editorial, high-contrast

---

## Colors

- Background: `#0d0d0d` (deep black)
- Surface: `#1a1a1a`
- Surface elevated: `#262626`
- Primary accent: `#00FF80` (neon green) — use sparingly
- Text primary: `#ffffff`
- Text secondary: `#a0a0a0`
- Text tertiary: `#666666`

**Rules:**
- Never use mid-gray backgrounds
- Accent color only for CTAs, highlights, links
- Maintain high contrast at all times

## Typography

- Headlines: Bold sans-serif, 4-5rem hero / 2.5-3rem sections
- Body: 400 weight, 1rem
- Labels: 0.75-0.875rem
- Tight letter-spacing on large headlines
- Generous line-height for body readability

## Layout

- Section padding: 80-120px vertical
- Content max-width: 1400px
- 12-column grid
- Tight component spacing: 16-24px
- Wide desktop margins: 80-120px

## Components

### Buttons
- Primary: Solid white or accent, black text
- Secondary: Outlined, white border, white text
- Hover: Background shift + slight scale

### Cards
- Large edge-to-edge imagery
- Category tags (small, rounded)
- Hover: scale or opacity transform

### Navigation
- Minimal, sticky
- Logo + links + CTA
- Transparent on hero, solid on scroll

### Sections
- Hero: Large headline + subheadline + CTA + full-width bg
- Stats/Trust: Client logos, award badges
- Services: 3-4 col grid, icon + title + desc
- Case studies: Large image + title + category, 2-3 col
- Testimonials: Profile photo + quote + name/role
- Process: Numbered steps (01, 02, 03...)
- FAQ: Accordion expand/collapse

## Imagery

- Full-width product mockups
- 16:9 aspect ratio for case studies
- Minimal line icons, accent colored
- Monochrome award/partner badges
- No generic stock photo aesthetics

## Motion

- Fade-in on scroll
- Smooth scroll behavior
- Subtle hover transforms (scale, opacity)
- Button hover: background shift + scale

## Do

- Deep black backgrounds
- Large bold typography
- Full-width hero sections
- High-quality imagery
- Gradient CTAs
- Minimal navigation
- Generous whitespace

## Don't

- Mid-gray backgrounds
- Overcrowded content
- Too many colors
- Generic gradients
- Stock photo aesthetics
